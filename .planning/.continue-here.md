## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Assuming http_client headers propagate | The openai SDK constructs its own header set and ignores most headers on a passed `http_client` | blocking | Always use `default_headers` or `default_query` for provider-required headers |
| Testing payload fixes without wire inspection | It's easy to believe a fix works when `_get_request_payload` shows the field, but the SDK or httpx might still strip it | advisory | Monkey-patch `client._post` or use an HTTP proxy to verify the actual bytes on the wire |
